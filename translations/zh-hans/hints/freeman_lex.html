首先，你需要if-else 语句来返回数字或 0，这取决于 list 的空性。用len() 检查 list 的空性 - 结果不等于 0。然后，用sum() 乘以 list 中的最后一个值。它的索引为len(list)-1 。在 sum() 中，使用list comprehensions 创建新的列表：使用range() 遍历给定的数组，步骤 2 只得到偶数索引，并用这些索引取列表中的值。

就在这里： 

def checkio(array: list[int]) -> int:

    if len(array)!= 0:
        sumEven = sum([array[i] for i in range(0, len(array), 2)])
        last = array[len(array) - 1]
        return sumEven * last 
    else:
        return 0

为了使代码更 "pythonic"（我建议你也去google一下这个词）和更短，使用try-except 来捕获IndexError，用slicing代替list comprehensions和list的负索引： 

def checkio(array: list[int]) -> int:
            
    try:
        return array[-1] * sum(array[::2]) 
    except IndexError:
        return 0
