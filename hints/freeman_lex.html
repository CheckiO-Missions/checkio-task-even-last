Here it is: 

def checkio(array: list[int]) -> int:

    if len(array) != 0:
        sumEven = sum([array[i] for i in range(0, len(array), 2)])
        last = array[len(array) - 1]
        return sumEven * last 
    else:
        return 0

To make the code more 'pythonic' (I advise you to google this term as well) and shorter, use try-except to catch IndexError, slicing instead of list comprehensions and negative indexes of list: 

def checkio(array: list[int]) -> int:
            
    try:
        return array[-1] * sum(array[::2]) 
    except IndexError:
        return 0
